# Example Reports

This directory contains sample Tier 1 (weekly) reports generated by SecIntel AI. These examples demonstrate the output format and content structure for each tracker.

## Available Examples

| Report | Tracker | Description |
|--------|---------|-------------|
| [defender_weekly_report.html](defender_weekly_report.html) | `defender` | Microsoft Defender product updates |
| [microsoft_products_weekly_report.html](microsoft_products_weekly_report.html) | `microsoft_products` | Microsoft security products (Entra, Intune, Purview, etc.) |
| [threat_intel_weekly_report.html](threat_intel_weekly_report.html) | `threat_intel` | Threat intelligence from security blogs and advisories |
| [llm_news_weekly_report.html](llm_news_weekly_report.html) | `llm_news` | LLM/AI provider updates |

## Report Features

Each report includes:

- **Executive Summary** - AI-generated overview of key updates
- **Article Summaries** - Individual summaries for each scraped article
- **Source Attribution** - Links to original sources
- **Categorization** - Articles grouped by product/topic
- **IOC Extraction** - Indicators of Compromise (threat_intel tracker)

## Generating Your Own Reports

```bash
# Generate a Tier 1 (weekly) report
python secintel.py --tracker defender --report --tier 1

# Generate reports for all trackers
python secintel.py --report --tier 1
```

## Report Tiers

| Tier | Time Window | Use Case |
|------|-------------|----------|
| 0 | 1 day | Daily digest |
| 1 | 7 days | Weekly summary |
| 2 | 14 days | Bi-weekly review |
| 3 | 30 days | Monthly archive |
